# COA Dashboard Demo

A static GitHub Pages demo for a Certificate of Analysis workflow dashboard.

## Features
- SAP/production-to-COA workflow visualization
- Role-based views for Primary QC, Backup QC, and Superusers
- Plant, status, and search filters
- Reminder, escalation, urgency, and submission tracking
- Manual COA request form
- Local browser persistence using localStorage
- Responsive layout with horizontally scrollable table

## Publish with GitHub Pages
1. Create a new GitHub repository.
2. Upload `index.html` to the repository root.
3. Open **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root`.
6. Save and wait for GitHub to publish the site.

No server, database, or external library is required for this demo.
